Documentation for the
ASxxxx Assemblers and
ASlink Relocating Linker

V3.00 - May 2000

The ASxxxx Cross Assembler and Linker package contains cross assemblers for the 6800(6802/6808), 6801(hd6303), 6804, 6805, 68HC08, 6809, 68HC11, 68HC12, 68HC16, 8051, 8085(8080), z80(hd64180), H8/3xx, DS8xCxxx, and 6500 series microprocessors. Complete source code is provided with the assembler/linker package.

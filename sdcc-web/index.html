Small Device C Compler (sdcc)

A portable C compiler for the Intel mcs51 series (8051, 8032...) and for the Zilog Z80. sdcc is reasonably easy to port to other eight bit processors.

This page is just a holder. sdcc is currently under development - the mcs51 port is stable, and the Z80 port generates valid if ugly code.

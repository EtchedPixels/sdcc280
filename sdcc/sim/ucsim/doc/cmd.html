Command reference of μCsim

General commands and information
  • conf Configuration
    • conf objects
  • help,? Help about commands
  • quit Quit
  • kill Shut down the simulator
  • exec Execute commands from file
  • expression Compute value of an expression
  • show Show different kind of information
    • show copying
    • show warranty
    • show option
    • show error
  • get Get various things
    • get option
    • get sfr
    • get error
  • set Set various things
    • set option
    • set error
    • set console
    • set memory
    • set bit
    • set hardware
  • state State of simulator
  • reset Reset
  • info Information
    • info breakpoints
    • info registers
    • info hardware
    • info memory
    • info variables
  • timer Managing timers to measure execution time and profiling
    • timer add
    • timer delete
    • timer get
    • timer start
    • timer stop
    • timer set
Memory manipulation
  • file,load Load FILE into ROM
  • download,dl Download (intel hex) data into ROM
  • fill Fill memory region with data
  • where,Where Search for data in memory
  • memory Manage address spaces, memory chips and address decoders
    • memory create
      • memory create chip
      • memory create addressspace
      • memory create addressdecoder
      • memory create banker
      • memory create bank
      • memory create bander
    • memory info
    • memory cell
  • var
Execution
  • r,run,go Go
  • stop Stop
  • s,step Step
  • n,next Next
  • pc Set/get PC
Breakpoints
  • break Set fix breakpoint
  • tbreak Set temporary breakpoint
  • clear Clear fix breakpoint
  • delete Delete breakpoint(s)
  • commands Define breakpoint script
Dump commands
  • dump Dump memory or bits
  • disassemble Disassemble
  • dc Dump code in disass format
  • dch Dump code in hex format
  • di Dump Internal RAM
  • dx Dump External RAM
  • ds Dump SFR
Removed obsolete commands
  • wi,Wi Search for string in Internal RAM (use where,Where instead)
  • wx,Wx Search for string in External RAM (use where,Where instead)
  • wc,Wc Search for string in ROM (use where,Where instead)
  • gopt Get value of option(s) (use get option instead)
  • sopt Set value of option (use set option instead)
  • g Go (use run instead)
  • bs Set FETCH Breakpoint (use break or tbreak instead)
  • bse Set EVENT Breakpoint (use break or tbreak instead)
  • bd Delete FETCH Breakpoint (use clear instead)
  • bde Delete EVENT Breakpoint (use delete instead)
  • ba Delete all breakpoints (use delete instead)
  • bl List Breakpoints (use info breakpoints instead)
  • dr Dump registers (use info registers instead)
  • dp Dump ports (use info hardware instead)
  • si Set Internal RAM (use set memory instead)
  • sx Set External RAM (use set memory instead)
  • sc Set code (ROM) (use set memory instead)
  • ss Set SFR area (use set memory instead)
  • sb Set bit (use set bit instead)
  • fi Fill IRAM area with data (use fill instead)
  • fx Fill XRAM area with data (use fill instead)
  • fs Fill SFR area with data (use fill instead)
  • fc Fill ROM area with data (use fill instead)
  • db Dump bit (use dump instead)
  • sp Set port pins (use set port instead)
